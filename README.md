## Projetos em destaque

### Chat Bot Pref
Plataforma de atendimento institucional com IA aplicada, arquitetura tenant-aware, RAG, guardrails, auditoria, observabilidade, CI/CD e deploy em AWS com Terraform.

É um dos principais cases do meu portfólio para demonstrar engenharia de IA com foco em arquitetura, rastreabilidade, operação e evolução para LLMOps.

### dev-workspace
Workspace operacional criado para padronizar ambiente, onboarding, governança, automação e execução técnica no dia a dia.

Reúne playbooks, validações locais, rotinas operacionais, linting, shift-left security e guardrails para uso de agentes de IA com mais contexto, previsibilidade e controle.

### AWS Web Infrastructure with Terraform
Projeto de infraestrutura como código para provisionamento de ambiente web na AWS com Terraform, modularização, backend remoto, validação automatizada e deploy controlado.

Demonstra organização de infraestrutura, previsibilidade operacional e disciplina de entrega.

### Observabilidade com Grafana, Prometheus e Loki
Stack de observabilidade com métricas, logs estruturados, dashboards operacionais e suporte a troubleshooting.

Foi construída para demonstrar monitoramento, leitura operacional, resposta a incidentes e maturidade na operação de sistemas distribuídos.

### API com CI/CD, Testes e Containerização
Projeto de fundação DevOps com FastAPI, testes automatizados, Docker, GitHub Actions, versionamento e pipeline de entrega.

Representa a base de disciplina de engenharia aplicada a backend, automação e qualidade.
